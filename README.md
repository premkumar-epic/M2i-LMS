# M2i-LMS

A modern Learning Management System built with Node.js, Express, Next.js, and Prisma.

## Project Structure

- `backend/`: Express.js server with TypeScript and Prisma.
- `frontend/`: Next.js application with Tailwind CSS.
- `docs/`: Comprehensive project documentation, API endpoints, and architectural diagrams.
- `features/`: Detailed functional specifications for all modules.

## Getting Started

Refer to the documentation in the `docs/` directory for onboarding and setup instructions:

- [Developer Onboarding Guide](docs/Developer_Onboarding_Guide.md)
- [API Documentation](docs/API_Endpoints.md)
- [Database Schema](docs/Database_Schema.md)

## Development Workflow

- `main`: Stable production-ready code.
- `develop`: Active development and integration branch.
- Feature branches should be branched from `develop`.
